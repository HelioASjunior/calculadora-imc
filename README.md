🧮 Calculadora de IMC
Uma aplicação web simples e funcional para calcular o Índice de Massa Corporal (IMC). Este projeto foi desenvolvido para praticar a manipulação do DOM com JavaScript e estilização básica com CSS.

🚀 Demonstração
A aplicação permite que o usuário insira seu peso e altura para receber instantaneamente seu IMC e a classificação correspondente (Abaixo do peso, Peso normal, Sobrepeso, etc.).


🛠️ Tecnologias UtilizadasHTML5: Estruturação dos elementos.CSS3: Design responsivo e estilização visual.JavaScript: Lógica de cálculo e interatividade.

📋Funcionalidades
[x] Entrada de peso (kg) e altura (m).
[x] Validação simples de campos.
[x] Cálculo automático ao clicar no botão.
[x] Exibição da categoria do IMC segundo a tabela da OMS.
